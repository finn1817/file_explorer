## 🎯 Quick Start Guide

### How to Run the App

1. **Navigate to the project:**
   ```
   cd C:\Users\danny\Desktop\MyApps\File-Explorer\app-code
   ```

2. **Run using the launcher (easiest):**
   - Double-click `run.bat`
   
3. **Or run from terminal:**
   ```
   cd python
   python main.py
   ```
